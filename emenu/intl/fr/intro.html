C'est quoi le Hi-Tunes?

Êtes-vous fatigué d'écouter des tonalités ennuyeuses quand vous appelez quelqu'un? Voulez-vous jouer une musique agréable ou une chanson à vos amis ou à vos proches quand ils vous appellent? Voulez-vous impressionner une personne chère à vous quand elle vous appelle? Hi-Tunes peut faire de vos rêves une réalité! En s'inscrivant au service Hi-Tunes, vous pouvez réserver à vos amis et à vos client clients une surprise! Joignez le monde Hi-Tunes MAINTENANT!
Comment puis-je m'inscrire au service Hi-Tunes?

Pour vous inscrire au service Hi-Tunes, composez le 666 ou visitez le site Web du service Hi-Tunes. Votre mot de passe de service Hi-Tunes est également le même mot de passe pour ouvrir une session dans site internet du service Hi-Tunes. Sur le site Web, vous pouvez changer, éditer et définir vos tonalités de retour d'appel personnelles.
Puis-je utiliser le Hi-Tunes?

Si vous êtes un abonné prépayé ou postpayé, vous pouvez vous inscrire au service Hi-Tunes en ligne via notre site Internet ou en appelant notre service client, ainsi vous pouvez profiter du service Hi-Tunes.
Comment puis-je l'activer?

Composez le 666 et suivez les instructions du guide. Une fois inscrit vous obtiendrez cinq sonneries gratuitement. Vous pouvez définir un numéro d'appelant pour chaque tonalité de retour d'appel, et l'appelant entendra la tonalité en vous appelant.

Vous pouvez également visiter le site web du service Hi-Tunes pour mieux voir les tonalités de retour d'appel. On vous fournit différents types de musiques telles que la musique pop, la musique classique, et à ainsi de suite
Librairie

Il y a différentes tonalités de retour d'appel dans la librairie de Hi-Tunes. Vous pouvez prévisualiser les tonalités de retour d'appel et rechercher votre musique préférée par le code, le nom ou l'artiste de la tonalité. Pour découvrir la musique la plus populaire, vous pouvez vérifier le classement mensuel, le classement hebdomadaire, et le classement total.
Mes Hi-Tunes

Sur la page Internet de Mes Hi-Tunes, vous pouvez exécuter plusieurs fonctions, comme par exemple éditer les tonalités de retour d'appel dans votre librairie personnelle, définir votre groupe de tonalités de retour d'appel personnelles, et définir une tonalité de retour d'appel par défaut.
Après que vous créiez un groupe de tonalités de retour d'appel, vous pouvez ajouter des tonalités à partir de votre librairie personnelle. Après avoir créé le groupe de Hi-Tunes avec succès, vous pouvez définir un numéro ou un groupe de numéros d'appelants pour jouer les tonalités de retour d'appel du groupe de Hi-Tunes d'une manière aléatoire.
Vous pouvez personnaliser un groupe de de Hi-Tune pour un numéro ou pour un groupe de numéros d'appelants. Quand le numéro d'appelant est un membre du groupe de numéros d'appelants, le système jouera la tonalité de retour d'appel spécifiée pour ce groupe de numéros. Si vous spécifiez un groupe de Hi-Tunes à un groupe de numéros, le système jouera les tonalités de retour d'appel d'une manière aléatoire.
La fonction de la liste noire vous est fournie pour empêcher des appels malveillants ou gênants. Si un numéro d'un appelant est ajouté à la liste noire, le système rejettera les appels venant de ce numéro.
Paramètres personnels  

Dans vos Paramètres personnels, vous pouvez modifier votre mot de passe, éditer les informations personnelles, suspendre le service, éditer les favoris et se désinscrire du service Hi-Tunes. Le mot de passe de Hi-Tunes est exigé à l'ouverture de session dans le système Hi-Tunes et pour accéder au système de gestion de voix. Vous pouvez changer votre mot de passe en ligne via le site web ou en composant le 2728.
Qu'arrive t-il si j'oublie de définir une tonalité?  

Pas vraiment important, le système a défini une tonalité de retour d'appel pour vous. Si vous oubliez de définir votre Hi-Tunes pour les appels d'arrivée le système appliquera les paramètres de Hi-Tunes par défaut pour vous.
Combien est-ce que ça coûte?

Hi-Tunes est service gratuit pour les premiers xxx mois. Après cette période vous payerez seulement 1 dinars par mois. Il y aura également des frais à payer pour chaque commande de tonalités additionnelles. Les prix des tonalités sont affichés dans la page web de la librairie de Hi-Tunes.
Comment puis-je me désinscrire du service Hi-Tunes?

C'est simple. Simplement composez le 666, ou visitez le site web du service Hi-Tunes pour vous désinscrire du service. Si vous voulez vous désinscrire du service pendant un certain mois, cette désinscription entrera en vigueur le début du mois suivant. Vous continuerez à payer les frais d'abonnement du mois courant.
